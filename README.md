📊 LR vs RF – React App
A simple React application to compare Linear Regression, Random Forest, and K-Nearest Neighbors (KNN) models through visualizations and predictions.

🚀 Features
Compare LR, RF & KNN predictions
Interactive inputs
Graph-based visualization
Clean and responsive UI

🛠️ Tech Stack
React.js
CSS / Tailwind
Chart library (Chart.js / Recharts)
Python (Backend for ML models)

⚙️ Setup
git clone https://github.com/sj1304/Linear-Vs-Random-forest-Comparison-Webapp.git
cd Linear-Vs-Random-forest-Comparison-Webapp
npm install
npm start

Open another terminal:
python backend.py

Runs on: http://localhost:3000

📊 Overview
Linear Regression: Works well for linear data
Random Forest: Handles complex, non-linear patterns
K-Nearest Neighbors (KNN):
Classification-based algorithm
Predicts based on closest data points
Useful for pattern recognition and grouping
